# Renovera Energia Solar

Landing page publica da Renovera para energia solar, dimensionamento fotovoltaico, simulacao de economia e edicao rapida de conteudo.

## Tecnologias

- React
- TypeScript
- Vite

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publicacao

Este repositorio publica automaticamente no GitHub Pages pelo workflow em `.github/workflows/deploy-pages.yml`.

URL publica:

```txt
https://renovera1.github.io/renovera-energia-solar/
```
